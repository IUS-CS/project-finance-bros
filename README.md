# Statement Reader

Statement Reader is a web application built with Django that allows users to view their credit card statements, visualize their financial history, and browse other available credit cards.

## Installation & Setup

Set up the virtual environment.

**Windows:**

```bash
python -m venv venv
```
**macOS & Linux:**

```bash
python3 -m venv venv
```

Activate the virtual environment.

**Windows:**

```bash
venv\scripts\activate.bat
```
**macOS & Linux:**

```bash
source venv/bin/activate
```

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install [Django](https://www.djangoproject.com/).

```bash
pip install django
```

Clone the GitHub repository.

```bash
git clone https://github.com/IUS-CS/project-finance-bros.git
```

Run the server.

**Windows:**

```bash
python project-finance-bros\src\manage.py runserver
```
**macOS & Linux:**

```bash
python3 project-finance-bros/src/manage.py runserver
```

The website is now running on your local machine, which can be found at [localhost:8000](http://localhost:8000).
